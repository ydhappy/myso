# 에고무기 보강 및 확장 제안

현재 구현은 Java 8 / 기존 무기 유지 / type2 변형 없음 / DB 최소 구조 기준입니다.

## 1. 즉시 반영 완료된 보강

```text
장르대화 전용 파일 EgoGenreTalk.java 분리
장르목록/대사목록 안내 EgoGenreGuide.java 추가
EgoTalk.chat()에서 장르대화 우선 라우팅
장르대화 전용 1.2초 딜레이 추가
기존 상태/조언/공격/멈춤 대화와 충돌 방지
예의/예의반대 말투 모두 지원
실제 작품 대사 복사 금지, 오리지널 대사만 사용
```

## 3. 다음 확장 제안

### 3.1 DB 대사팩화

현재 대사는 Java 배열에 있습니다. 추후 운영 중 리로드로 대사를 바꾸려면 DB화가 좋습니다.

추천 테이블:

```sql
CREATE TABLE ego_talk_pack (
    id BIGINT NOT NULL AUTO_INCREMENT,
    genre VARCHAR(30) NOT NULL,
    tone VARCHAR(30) NOT NULL,
    keyword VARCHAR(80) NOT NULL DEFAULT '',
    message VARCHAR(255) NOT NULL,
    use_yn TINYINT(1) NOT NULL DEFAULT 1,
    PRIMARY KEY (id),
    INDEX ego_talk_pack_idx (genre, tone, use_yn)
) ENGINE=InnoDB DEFAULT CHARSET=euckr COLLATE=euckr_korean_ci;
```

효과:

```text
서버 재시작 없이 대사 수정 가능
이벤트 기간 대사 추가 가능
운영자별 서버 분위기 커스터마이징 가능
```

### 3.2 호감도/유대감 시스템

에고와 대화하거나 사냥하면 유대감이 오르게 할 수 있습니다.

예시:

```text
대화 1회 +1
레벨업 +10
사망 위기 생존 +20
에고삭제 시 초기화
```

활용:

```text
유대감 높을수록 대사 부드러움
유대감 높을수록 경고 메시지 빠름
유대감 특정 수치에서 특별 대사 해금
```

### 3.3 상황별 자동 대사

현재는 일반채팅 호출 중심입니다. 추후 자동 상황 대사를 늘릴 수 있습니다.

추천 상황:

```text
HP 30% 이하
MP 20% 이하
선공 몬스터 접근
보스 감지
에고 레벨업
스턴 성공
반격 성공
장시간 사냥
장시간 미전투
귀환 직후
사망 직전
```

### 3.4 대사 반복 방지

현재 랜덤 선택만 있습니다. 최근 N개 대사를 저장해 같은 대사를 피하면 자연스러워집니다.

구현 방식:

```text
Map<Long, LinkedList<String>> recentTalkMap
캐릭터별 최근 5개 메시지 저장
랜덤 선택 시 최근 메시지 제외
전부 제외되면 초기화
```

### 3.5 에고 성격 확장

현재 말투는 예의/예의반대 2개입니다. 추후 성격을 확장할 수 있습니다.

후보:

```text
예의
예의반대
냉정
다정
광전
현자
겁쟁이
수다쟁이
무협풍
영화풍
웹툰풍
```

단, 성격을 많이 늘리면 대사 관리가 복잡해지므로 DB 대사팩화 후 확장하는 것이 좋습니다.

### 3.6 클라이언트 표시 강화

현재 짧은 답변은 말풍선, 긴 답변은 egoletter 편지창 기준입니다.

추가 가능:

```text
에고 대사 색상 구분
위험 대사 빨간색
조언 대사 노란색
잡담 대사 하늘색
장르 대사 보라색
```

### 3.7 운영자 테스트 명령

대사 테스트용 명령을 추가하면 검수가 쉬워집니다.

예시:

```text
.에고대사테스트 드라마 예의
.에고대사테스트 웹툰 예의반대
.에고대사목록
```

## 4. 주의사항

```text
실제 드라마/영화/웹툰 대사 복사 금지
유명 작품명 직접 패러디 남발 금지
욕설/비하 강도 과도하게 올리지 않기
전투 중 대사 스팸 방지 필수
DB 대사팩화 전까지 Java 파일이 너무 커지지 않게 장르별 분리 유지
```
