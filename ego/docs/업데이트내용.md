### 유지하는 분리 테이블 기준

```text
ego_skill_base
- 무기별 상태가 아니라 전체 능력 기본값이므로 분리 유지
- Java EgoWeaponAbilityController의 능력명과 1:1 유지

ego_log
- 발동 이력은 누적 데이터라 ego 본문에 병합하지 않음

ego_talk_pack
- 대사 콘텐츠는 다건 데이터라 ego 본문에 병합하지 않음

ego_config
- 운영 중 수치 조정을 위해 분리 유지

ego_level
- 레벨별 성장/전투 보정값이므로 분리 유지

ego_item_template
- 서버별 item 테이블 구조가 달라 참고 템플릿으로만 유지
```

## 운영 주의

전체 삭제/전체 초기화 SQL은 제공하지 않습니다.

에고 삭제 명령도 제공하지 않습니다. 운영자가 수동으로 정리해야 할 경우에는 반드시 대상 item_id를 확인한 뒤 DB에서 개별 정리해야 합니다.

에고 구슬 실제 아이템 등록은 서버별 item 테이블 구조에 맞춰 `ego_item_template` 값을 참고해 수동 등록합니다.
