# 에고무기 업데이트내용

## 2026-04-28 단순 재정립

### 실제 구조

```text
SQL 파일: ego/sql/ego_schema.sql 1개
문서 파일: ego/docs/사용방법.md, ego/docs/업데이트내용.md 2개
빠른 안내: ego/README.md
Java 배포: ego/java
HTML 템플릿: ego/html/egoletter.htm
```

### 핵심 정책

```text
점명령 생성/삭제/변경 제거
에고 생성은 에고 구슬 아이템 사용으로만 처리
에고 구슬은 착용 무기에 에고가 없을 때만 최초 생성
이미 에고무기면 능력/대화/레벨/경험치 변경 없음
에고무기가 다른 캐릭터에게 이동되면 현재 캐릭터를 주인으로 자동 재인식
무기 타입별 허용/차단 규칙 제거
무기 슬롯 아이템이면 에고 대상, 낚싯대만 제외
```

### 능력명 정책

```text
Java 전투 컨트롤러 런타임 능력명과 DB 저장 능력명을 1:1로 유지
단순명 입력은 EgoWeaponDatabase에서 런타임 능력명으로 정규화
```

### 대화 성향 정책

```text
Java/DB 성향값: 예의 / 예의반대 / 싸이코패스
DB 저장 컬럼: ego.ego_type, ego_talk_pack.tone
구슬 최초 생성 랜덤 성향: 예의 / 예의반대 / 싸이코패스
```

정규화 alias:

```text
반말 / 막말 / 싸가지 -> 예의반대
사이코패스 / psycho / psychopath -> 싸이코패스
그 외 미지정값 -> 예의
```

`싸이코패스`는 게임 내 무기 인격용 차갑고 불안정한 분위기 말투입니다. 실제 위협, 자해, 잔혹행동 지시를 목적으로 하지 않습니다.

## 운영 주의

전체 삭제/전체 초기화 SQL은 제공하지 않습니다.

에고 삭제 명령도 제공하지 않습니다. 운영자가 수동으로 정리해야 할 경우에는 반드시 대상 item_id를 확인한 뒤 DB에서 개별 정리해야 합니다.

에고 구슬 실제 아이템 등록은 서버별 item 테이블 구조에 맞춰 `ego_item_template` 값을 참고해 수동 등록합니다.
